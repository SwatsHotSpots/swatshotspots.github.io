Welcome to Swat's Hot Spots

Discover if your favorite Study Space is occupied!

What We Do

  • Allow Swarthmore college students to have a better idea of available study space and seating and allow them to check this without physically being in the space.
  • Used the Omron D6T Thermal sensor and configure the sensors to record data on the NodeMCU.
  • Transmit the data wirelessly using the NodeMCU's built-in ESP8266 WiFi module.
  • Process the data into a useable form and finally viewing the data in a way that is helpful to a user.

Software

We used the Arduino IDE primarily to load software onto the NodeMCU. Our code had several stages. The first was to read the thermal sensor and parse the temperature information. Using this data, we determined if a space was occupied. This information was them sent over WiFi to this website. The source code was programmed in C, however the website and the communication sent between the ESP8266 and our website is in HTML. The full code used to implement our project can be found on our Github
